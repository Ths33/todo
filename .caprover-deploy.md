# CapRover Deployment

Este arquivo contém informações sobre o deploy no CapRover.

## Arquivos criados

1. **Dockerfile** - Define como a aplicação será construída e executada
2. **captain-definition** - Arquivo de configuração do CapRover
3. **next.config.js** - Atualizado com `output: 'standalone'` para otimizar o build

## Como fazer deploy

### 1. Via CapRover CLI

```bash
# Instalar o CapRover CLI (se ainda não tiver)
npm install -g caprover

# Fazer login no servidor CapRover
caprover login

# Fazer deploy
caprover deploy
```

### 2. Via Git

Configure seu app no CapRover para fazer deploy automático via Git:

1. No painel do CapRover, vá para seu app
2. Ative "Enable App Deployment via Git"
3. Siga as instruções para adicionar o remote e fazer push

```bash
git remote add caprover [URL_DO_SEU_CAPROVER]
git push caprover main
```

### 3. Via tarball (upload manual)

```bash
# Criar tarball
tar -czf deploy.tar.gz .

# Fazer upload no painel do CapRover
```

## Variáveis de ambiente

Se você precisar de variáveis de ambiente, adicione-as no painel do CapRover em:
**App Configs** → **Environment Variables**

## Porta

O Next.js rodará na porta 3000 internamente. O CapRover fará o proxy automaticamente.
